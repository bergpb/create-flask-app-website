Create Flask App

On Linux run commands bellow: 

$ curl https://storage.googleapis.com/spacedevs_storage_intervalo_1/create-flask-app
> ~/.local/bin/create-flask-app
$ sudo chmod -R 777 ~/.local/bin/create-flask-app

and then execute the follows command for create a project using cli: 

$ create-flask-app myproject_name

For take create-flask-app for Windows 10 click here to download of binary. After and same folder, run the command to PowerShell for create a project: 

$ create-flask-app.exe myproject_name
